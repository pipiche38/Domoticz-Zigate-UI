Ce plugin permet d'utiliser une ZiGate avec le logiciel DomoticZ afin de pouvoir communiquer avec des objets utilisant le protocole ZigBee.

Retrouver toutes les informations pour utiliser ce plugin sur le Wiki du plugin .
Refer to the plugin Wiki for more information.

La petite icône renvoit vers les infromations sur le Wiki français.

En cas de problèmes

  • Vérifiez les logs de DomoticZ, ils contiennent de nombreuses informations utiles à la compréhension du(es) dysfonctionnements.

  • Venez en discuter sur les forums :
    • Le forum français en suivant le Guide pour bien formuler une demande afin que nous ayons un maximum d'informations dés le départ.
    • Le forum anglais

Venez discuter avec nous !

Nous sommes une petite équipe de passionnés qui partageons le plaisir d'utiliser DomoticZ avec la ZiGate.

Retrouvez-vous sur keybase.io (team Zigate) pour des discussions en temps réel.
